# BreathView
一款简单的呼吸控件

Add it in your root build.gradle at the end of repositories:


	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  Add the dependency
  
  
  	dependencies {
		implementation 'com.github.arpsyalin:BreathView:1.0'
	}
